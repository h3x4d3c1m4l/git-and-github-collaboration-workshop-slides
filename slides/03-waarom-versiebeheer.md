# Waarom Git?

- Goede softwareondersteuning
  - Brede OS ondersteuning
  - Veel clients
- Veel keuze uit services
  - Bekendst: GitHub, Bitbucket, Azure DevOps, GitLab, ...
- Goed gedocumenteerd
- Alternatieven:
  - CVS: Traag
  - SVN: Geen branches
  - Microsoft TFS: Ingewikkeld, niet cross platform
  - Mercurial: Goed, minder populair
