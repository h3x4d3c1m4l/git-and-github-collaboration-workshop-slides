# Niet op Git -  .gitignore

- Bestand, meestal in de hoofdmap
- Heet altijd `.gitignore`
- Bevat uitsluitingen:
  - Bestandsnamen
  - Mappen
  - Patronen (bijvoorbeeld `*.exe`, `*.dll`)
  - Uitzonderingen op de regels
- Soms aangemaakt door je IDE
- Kan met GitHub aangemaakt worden
